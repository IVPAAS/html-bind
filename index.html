Html-bind

Light code for basic html attribute bind to javascript scope.

View project onGitHub

Why?

  • HTML attribute binding is good for isolate UI&Logic.
  • Save code for update UI.
  • State driving architecture.
  • Light code for performance critical & light applications.

How?

  • Add the attribute you want to watch.
  • Connect the jQuery function you want to execute.
  • List the javascript objects you want to watch.
  • Let the kWatcher know all the data and that it!.

Example:

<div kshow="playing" khide="!playing" data-value="data">
Playing
</div>

<input type="text" onkeyup="$kscope.data=this.value">

<div data-value="data" format="addSomthing">Loading...</div>


<button value="Play" onclick="$kscope.playing = true;">Play</button>
<button value="Pause" onclick="$kscope.playing = false">Pause</button>

<script>
    var $kscope = {
        playing:true,
        data:"content123",
        addSomthing:function(value){
            return value  +"123";
        }
    };


    var watcher = new kWatcher($kscope);

    watcher.bind("playing",function(prop,old,newVal)
    {
        console.log(prop +":" + old +"--" + newVal);
    })
</script>
